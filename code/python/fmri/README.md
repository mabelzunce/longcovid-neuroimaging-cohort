# fMRI overlap check

This folder contains a notebook to visualize raw vs preprocessed fMRI images and check
alignment with AAL3 and Schaefer2018 templates.

## Files
- `fmri_overlap_check.ipynb`: Loads the raw and preprocessed fMRI images, resamples the templates,
  and overlays them for visual inspection.
- `requirements.txt`: Minimal Python dependencies for the notebook.

## Quick start
Open the notebook in Jupyter and run the cells top to bottom. If files are moved,
update the paths in the **Input paths** cell.
